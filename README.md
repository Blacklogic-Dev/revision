📘 Africa Institution of Technology – Revision Exam System
🚀 Overview
The AIT Revision Exam System is a lightweight, frontend-based web application designed to help students prepare for their final examinations through structured revision assessments.
This system provides:
Randomized questions
Instant auto-marking
Performance feedback
Firebase data storage for monitoring
It is built using HTML, CSS, JavaScript, and Firebase Realtime Database.
🎯 Purpose
The Revision Exam System is designed to:
Reinforce key concepts across all courses
Identify weak areas before final exams
Simulate real exam conditions
Improve student confidence and performance
📚 Courses Covered
Each revision exam contains 20 selected questions based on the most important and challenging topics:
💻 Frontend Development
🏗 Construction Technology
🎨 Logo Design
🧠 Administration Technology
